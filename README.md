## 📖 About

**MeddyNet** is a modern healthcare platform designed to simplify the process of finding and booking diagnostic tests across India. Our platform connects patients with NABL/ISO certified pathology labs, enabling transparent price comparison, home sample collection booking, and secure digital health report storage.

> 🎯 **Mission**: Make quality diagnostic healthcare accessible, affordable, and convenient for every Indian household.

---

## ✨ Features

### 🔬 Compare Labs
Find nearby pathology labs and compare test prices instantly with real-time availability and ratings.

### 🏠 Home Sample Collection
Book certified lab technicians to collect samples from your doorstep at your convenience.

### 📍 Nearby Labs Map
Interactive map interface to discover and locate trusted diagnostic labs in your vicinity.

### 📄 Digital Health Reports
Secure health reports stored permanently in your MeddyNet account, accessible anytime, anywhere.

### 🔍 Smart Test Search
Intelligent search with auto-suggestions for diagnostic tests, health packages, and lab profiles.

### ⭐ Verified Reviews
Authentic patient ratings and reviews to help you choose the best lab for your needs.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tools required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/arpit0381/MeddyNet.git
   ```

2. **Navigate to the project**
   ```bash
   cd MeddyNet
   ```

3. **Open in browser**
   ```bash
   # Simply open the index.html file in your browser
   # Or use a local server:
   npx serve .
   ```

4. **That's it!** 🎉 The landing page will load with all animations and interactions.
